# hotel-management-system-java
Proyecto académico desarrollado en Java para la materia Progr. Orientada a Objetos. El sistema permite gestionar reservas, clientes y habitaciones de un hotel, integrando una base de datos local (MySQL) y una interfaz gráfica (Swing). Incluye aplicación de los principios de POO, manejo de excepciones, persistencia de datos y arquitectura modular.
